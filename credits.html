Développement

Ce projet a été réalisé dans le cadre d'un mini-projet académique à SUPINFO par des étudiants passionnés de développement web.

Technologies utilisées :
• HTML, CSS, JavaScript
• Responsive Design
• API REST (OMDb)

Sources de données

OMDb API

Toutes les informations sur les films et séries proviennent de l'OMDb API, une API RESTful gratuite pour obtenir des informations sur les films.
