Développement

Ce projet a été réalisé dans le cadre d'un mini-projet académique à SUPINFO par des étudiants passionnés de développement web.

Technologies utilisées :
• HTML5, CSS3, JavaScript (ES6+)
• Responsive Design (Mobile-First)
• API REST (OMDb)

Sources de données

OMDb API

Toutes les informations sur les films et séries proviennent de The Open Movie Database (OMDb API), une API RESTful gratuite pour obtenir des informations sur les films.

→ Visiter OMDb API
